# Convertisseur
Convertisseur de nombres avec bases différentes, développé en Symfony 4 et orienté TDD

### Installation

- Installation des dépendances:
```bash
composer install
```

- Lancer server Symfony :
```bash
php bin/console server:run
```

### Tests

- Lancer les tests :
```bash
php bin/phpunit --testdox
```
